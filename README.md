# ☁️ CloudMemo

Красивый Markdown редактор заметок с прозрачным glass-интерфейсом для Windows.

## Что умеет
- 📝 Markdown редактор с live preview
- 📋 Список заметок в боковой панели
- 🔍 Мгновенный поиск
- 💾 Автосохранение
- 🪟 Прозрачный glass UI с blur-эффектом
- ⚡ Быстрый и лёгкий (Rust + Tauri)

## Как скачать
Перейди в раздел [Releases] и скачай установщик.

## Как собрать самому
git clone https://github.com/wh1laye/CloudMemo.git
cd CloudMemo
npm install
cargo tauri dev
